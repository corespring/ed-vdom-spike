Selective dom updating. Instead of just setting the element's html to some string value, I want to update the dom based on the changes.
The end goal is that - calls to $compile can be minimized to only when they are needed.

Instructions

Click 'Add thing' button. This will add '<thing></thing>' to the markup. This will be picked up by the diff as an insert event.
A hook will then be added that will be called when the element is added to the dom.
Finally the hook will call $compile on the element.
